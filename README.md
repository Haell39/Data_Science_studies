# Curso Intensivo de Python para IA e Data Science

Bem-vindo ao curso intensivo de Python focado em Inteligência Artificial e Data Science!

## 📁 Estrutura do Curso

```
python-crashcourse-datascience/
├── README.md                    # Este arquivo
```

## 🎯 Objetivos do Curso

- Dominar os fundamentos de Python
- Aprender manipulação de dados com Pandas e NumPy
- Criar visualizações impactantes com Matplotlib e Seaborn
- Entender conceitos estatísticos essenciais
- Implementar algoritmos de Machine Learning
- Explorar Deep Learning com TensorFlow/Keras
- Desenvolver projetos práticos reais

## 🚀 Como Usar Este Curso

1. Siga a ordem dos módulos
2. Leia os arquivos teóricos primeiro
3. Pratique com os exemplos práticos
4. Resolva os exercícios propostos
5. Avance para o próximo módulo

## 📋 Pré-requisitos

- Computador com Python instalado (versão 3.8+)
- Editor de código (VS Code recomendado)
- Conhecimentos básicos de programação (opcional)

## 🛠️ Instalação das Bibliotecas

```bash
pip install numpy pandas matplotlib seaborn scikit-learn tensorflow jupyter notebook
```
