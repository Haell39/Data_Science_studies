## 🟢 BLOCO 2: Nível Iniciante (Estruturas Condicionais)
*Foco: Dominar if/else e tomada de decisão*
 
### 1. Classificador de Números
Escreva um algoritmo que receba um número inteiro e imprima se ele é:
- **Par** ou **Ímpar**
- **Positivo**, **Negativo** ou **Zero**
 
### 2. Cálculo de Média Acadêmica
Crie um programa que receba quatro notas de um aluno:
- Calcule a média aritmética
- Se média >= 7 → "Aprovado"
- Se média entre 5 e 6.9 → "Recuperação"
- Se média < 5 → "Reprovado"

## 🟡 BLOCO 3: Nível Intermediário (Estruturas de Repetição)
*Foco: Loops (for, while) e processamento de múltiplos dados*
 
### 3. Tabuada Personalizada
Peça ao usuário um número e gere a tabuada dele de 1 a 10.
Exemplo output: `5 x 1 = 5`
 
### 4. Somatório e Filtro
Crie um programa que peça ao usuário para digitar 10 números. Ao final, mostre:
- A soma total de todos os números
- Apenas os números maiores que 50
- O maior e o menor número informado

## 🔴 BLOCO 4: Nível Avançado (Listas e Lógica de Processamento)
*Foco: Trabalhar com estruturas de dados e algoritmos*
 
### 5. Verificador de Palíndromos
Receba uma palavra ou frase e verifique se é um palíndromo (lê-se igual de trás pra frente).
- Ignore espaços e maiúsculas/minúsculas
- Exemplo: "Arara" → True
 
### 6. Contador de Frequência
Dado um conjunto de dados (lista de nomes ou números), conte quantas vezes cada elemento aparece e exiba o resultado.

## 🧠 BLOCO 5: Fundamentos (Progressivo)
 
### Nível 1 — Base (Obrigatório)
 
**1️⃣ Maior número**
Dado dois números, retorne o maior.
 
**2️⃣ Par ou ímpar**
Crie uma função que recebe um número e retorna `"par"` ou `"ímpar"`
 
**3️⃣ Soma de 1 até N**
```python
soma(5) → 15  # 1+2+3+4+5
```
 
**4️⃣ Contar de 1 até N**
Imprima:
```
1
2
3
...
N
```
 
---
 
### Nível 2 — Loop + Condição
 
**5️⃣ Maior de uma lista**
```python
[4, 7, 2, 9, 1] → 9
```
 
**6️⃣ Contar pares em uma lista**
```python
[1,2,3,4,5,6] → 3
```
 
**7️⃣ Média de uma lista**
```python
[10, 20, 30] → 20
```

### Nível 3 — Começa a Ficar Interessante
 
**8️⃣ Número primo**
