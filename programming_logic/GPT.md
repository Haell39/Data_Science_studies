# 🧠 Nível 1 — Base (obrigatório dominar)

### 1️⃣ Maior número

Dado dois números, retorne o maior.

👉 tu já fez, só reforçar

---

### 2️⃣ Par ou ímpar

Crie uma função que recebe um número e retorna:

* `"par"` ou `"ímpar"`

---

### 3️⃣ Soma de 1 até N

Exemplo:

```python
soma(5) → 15  # 1+2+3+4+5
```

---

### 4️⃣ Contar de 1 até N

Imprima:

```
1
2
3
...
N
```

---

# ⚙️ Nível 2 — Loop + condição

### 5️⃣ Maior de uma lista

```python
[4, 7, 2, 9, 1] → 9
```

---

### 6️⃣ Contar pares em uma lista

```python
[1,2,3,4,5,6] → 3
```

---

### 7️⃣ Média de uma lista

```python
[10, 20, 30] → 20
```

---
