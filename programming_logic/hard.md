# Exercícios de Lógica de Programação — Nível Difícil

1) Merge Sort (recursivo)
   - `merge_sort(lista)` que implemente merge sort e retorne lista ordenada.

2) Subconjunto com soma alvo (subset sum) — backtracking
   - `subset_sum(lista, alvo)` que retorna True se existir um subconjunto cuja soma é `alvo`.

3) Implementar LRU Cache simples
   - Classe `LRUCache(capacidade)` com `get(key)` e `put(key, value)` (O(1) esperável).

4) Avaliador de expressões aritméticas simples
   - `avaliar(expressao)` que suporte `+ - * /` e parênteses (assuma tokens separados por espaços ou implemente tokenização simples).

5) Caminhamento em grafo: BFS e distância mínima
   - `bfs(grafo, inicio)` que retorna ordem de visita e `distancia_minima(grafo, inicio, alvo)`.

Observação: escreva testes com casos pequenos e documente complexidade esperada.
