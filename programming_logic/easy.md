# Exercícios de Lógica de Programação — Nível Fácil

1) Soma de lista
   - Escreva uma função `soma(lista)` que receba uma lista de números e retorne a soma.

2) Máximo e mínimo
   - Crie uma função `min_max(lista)` que retorne uma tupla `(min, max)` para a lista.

3) Contar pares
   - Função `conta_pares(lista)` que retorna quantos números pares existem na lista.

4) N-ésimo Fibonacci (iterativo)
   - `fibonacci(n)` que retorna o n-ésimo termo (assuma `fibonacci(0)=0, fibonacci(1)=1`).

5) Verificar palíndromo
   - `eh_palindromo(s)` que verifica se a string `s` é palíndroma (ignorando espaços e case).

Instrução: para cada exercício, escreva testes simples com exemplos e trate entradas inválidas de forma básica.
