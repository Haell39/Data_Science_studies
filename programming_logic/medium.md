# Exercícios de Lógica de Programação — Nível Médio

1) Verificador de primo
   - `eh_primo(n)` que retorna True se `n` for primo. Otimize até sqrt(n).

2) Ordenação sem built-in
   - Implemente `insertion_sort(lista)` que retorna a lista ordenada (não usar `sorted()`).

3) Frequência de palavras
   - `frequencia_palavras(texto)` que retorna um dicionário com contagem de cada palavra (ignore case e pontuação simples).

4) Busca binária (iterativa)
   - `busca_binaria(lista_ordenada, alvo)` retorna o índice do alvo ou -1.

5) Soma de pares com alvo
   - `encontra_par_com_soma(lista, alvo)` que retorna um par de índices cujo valores somam `alvo` (complexidade O(n)).

Desafio opcional: escreva casos de teste para entradas com valores repetidos e listas vazias.
