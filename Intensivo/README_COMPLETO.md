## 🎯 Módulo 1: Fundamentos Gerais (80/20)

### Objetivo
Dominar os 20% mais importantes para 80% do uso diário em Data Science.

- **Python Essencial**: Variáveis, estruturas, funções, POO básico
- **NumPy Básico**: Arrays, operações vetorizadas, indexação
- **Pandas Essencial**: Series, DataFrames, manipulação de dados
- **Visualização Rápida**: Gráficos essenciais com Matplotlib
- **ML Básico**: Regressão, classificação, métricas fundamentais

### Projetos Práticos
- **Análise de Dados Completa**: Dashboard de vendas com insights

## 📚 Como Usar Este Curso

### 🔄 Fluxo Recomendado

1. **Comece pelos Fundamentos** (01-fundamentos-gerais/)
   - Execute cada arquivo na ordem
   - Pratique com os projetos mini
   - Certifique-se de entender os conceitos

2. **Avance para o Essencial** (02-essencial-80-20/)
   - Aprofunde o conhecimento
   - Execute os projetos completos
   - Foco em aplicações reais

3. **Projetos Finais**
   - Escolha um projeto completo
   - Adapte para seus dados
   - Faça deploy

### 💡 Dicas de Estudo

- **Pratique Código**: Não apenas leia, execute tudo
- **Modifique e Experimente**: Altere parâmetros, veja o que acontece
- **Resolva Problemas**: Tente aplicar em seus próprios dados
- **Documente**: Anote suas descobertas e dificuldades
